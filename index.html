FAQ

What is this style guide for?

This style guide helps keep design and development consistent across projects by documenting reusable design elements and styles.

Can I customize the colors?

Yes! All colors are defined using CSS custom properties, making them easy to override or theme.

Where can I find the source code?

The style guide is built as a single HTML/CSS page and can be found at codepen.io/Kriesse/pen/Byaryzo.

Modal Dialog

This modal uses the HTML dialog element.

Modal Title

This is a native HTML dialog element. It’s great for simple UI dialogs and comes with great acessibility.

  • When dialog is open, user can not interact with the rest of the page
  • Positioning is done natively. The dialog always shows up in the center. Scroll position of the page is not affected.
  • No more div is needed to create the overlay. The dialog comes with a ::backdrop pseudo-element.
  • It comes with good default styles. Even if there is no CSS for some reason, it will look like a modal dialog with a backdrop.
